<h1>SITE DO ENEM</h1>
<h2>SOBRE</h2>
<p>Fiz esse projeto para facilitar a visualização das notas de corte do ENEM, sendo possível pesquisar por universidade, cidade e curso. Após selecionar o curso
desejado, o site mostra as informações de todas as notas de corte, sendo ampla concorrência ou não.</p>
<p>Como utilizei o Next.JS, consegui criar uma página individual para cada uma das opções(universidade, cidade e curso), sendo possível gera-las antecipadamente para ganhar poder de performance, transformando todas as páginas em páginas estáticas HTML/CSS/JavaScript</p>
<p>Porém para fazer isso, tive que primeiro ter todos esses dados agrupados em um só JSON, o que eu não tinha. Primeiramente, eu peguei todos esses dados manualmente e inseri-los no Excel. Após isso, criei um programa que cria JSONs personalizados para mim, dependendo do que eu precisar. Nesse caso, o programa analiza todas as informações e gera um único JSON com os dados, parecendo uma árvore familiar, onde todas as informações estão conectadas. Rodando o programa e conseguindo o JSON já tratado, pude continuar com o desenvolvimento do Site.</p>
